# Machine Dev

---

## Summary

Vagrant file for setup virtual machine for development

## Description

## Prerequisite

* Vagrant
* Virtual Box

## Useful commands

```bash
# Create (if the machine is not existed) and start it
vagrant up

# Shutdown the machine
vagrant halt

# Suspend the machine
vagrant suspend

# Resume the the machine from suspend state
vagrant resume

# Reload config from Vagrantfile into the machine
vagrant reload

# Re-run the provision
vagrant provision

# Destroy the machine
vagrant destroy
```
